# SQL-powered-Sales-intelligence-dashboard
Sales Analytics Dashboard built using SQL Server, Power BI, and Excel
## Project Overview

Developed an end-to-end Sales Analytics Dashboard using Microsoft SQL Server, Power BI, and Excel. The project analyzes sales performance across products, regions, and salespersons through interactive visualizations and KPI tracking.

## Tools & Technologies

* Microsoft SQL Server Express
* SQL Server Management Studio (SSMS)
* Power BI
* DAX
* Power Query
* Microsoft Excel

## Data Model

A Star Schema data model was implemented using:

### Dimension Tables

* Products
* Regions
* Salespersons

### Fact Table

* Sales

Relationships were established between the fact table and dimension tables to support efficient reporting and analysis.

## Key KPIs

* Total Revenue
* Total Profit
* Units Sold
* Profit Margin %
* Revenue Trends

## Dashboard Features

* Interactive KPI Cards
* Revenue Trend Analysis
* Product Performance Analysis
* Regional Performance Tracking
* Salesperson Performance Monitoring
* Dynamic Filters and Slicers
* Drill-Down Reporting
